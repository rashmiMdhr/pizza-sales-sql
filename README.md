Pizza Sales Analysis Using SQL
Project Overview

This project analyzes a pizza sales dataset using SQL to extract meaningful business insights. The analysis focuses on sales performance, customer preferences, revenue generation, and ordering patterns.

Key Insights
Large pizzas were the most frequently ordered size.
The Classic category contributed significantly to total sales.
Customer demand was highest during lunch and dinner hours.
A small number of pizza types generated a large portion of total revenue.
Revenue showed steady growth over time.
